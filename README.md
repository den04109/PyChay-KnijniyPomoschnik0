# 📚 PyChay — Книжный помощник

## 🧠 О проекте
Проект разработан командой **PyChay** из Ульяновского IT-куба.

Книжный помощник — это веб-сервис на Python, которое использует нейронную сеть для поиска ответа на вопросы пользователя в загружаемых книгах.

Основная идея проекта — создать **умный поисковик по содержимому книг**, который:

* 🔍 находит информацию внутри загруженных книг
* 🧠 анализирует текст с помощью ИИ
* ✨ при добавлении книги автоматически определяет название книги на основе её содержания



---

## 🚀 Основные возможности

* 📥 Загрузка книг на сайт с автоматическим определение названия
* 🔎 Поиск информации внутри текста этих книг в режиме чата
* 👤 Система регистрации и авторизации пользователей
* 📚 Работа с базой данных

## 📦 Установка и запуск

```bash
# Клонировать репозиторий
git clone https://github.com/den04109/PyChay-KnijniyPomoschnik0

# Создать venv
python -m venv venv    

# Установить зависимости
pip install -r requirements.txt

# Получите ключ для работы с Mistral (инструкцию прилагаем далее)

# Создайте в корневой папке проекта файл ".env"

# Поместите в этот файл следующую строчку:
 SECRET_KEY = "ваш полученный ключ"

# Запустите проект
python main.py
```

Перейдите по ссылке http://127.0.0.1:8083

##  Инструкция получения и внедрения ключа Mistral Ai

Инструкция получения и внедрения ключа Mistral:

 Шаг 1: Регистрация
1. Перейдите на сайт console.mistral.ai (https://console.mistral.ai/).
2. Создайте аккаунт. 
3. После регистрации подтвердите свой email (вам придет письмо со ссылкой).

 Шаг 2: Верификация
1. При первом входе система может попросить вас ввести номер телефона для верификации (защита от спама).

 Шаг 3: Создание ключа
1. В меню слева найдите раздел "API Keys".
2. Нажмите кнопку "Create new key".
3. Дайте ключу название (например, "MyProject") для удобства.
4. Важно: Скопируйте ключ сразу. После того как вы закроете окно, вы больше не сможете увидеть сам текст ключа, только его название. Если вы его потеряете, придется создавать новый.


Шаг 5. Внедрение ключа в проект

1. Используйте ключ как указано в инструкции по запуску проекта

## 📦  Работа с сайтом

Зарегистрируйтесь и авторизуйтесь на сайте.

Перейдите на страницу с чатом чтобы задать вопрос

Сайт отвечает только по тем книгам который загруженны в библиотеку.

Если подходящей для ответа книги в библиотеке нет, чат скажет что не может найти ответ.

Загруженные книги можно увидеть на странице "Библиотека" и также добавить свою книгу.

При добавлении своей книги вам будет предложена функция автоматического распознавания названия книжного произведения.

В библиотеку нельзя добавить книгу которая там уже есть.
